# OpenAI Integration - Quick Start Guide

## 🚀 Get Started in 5 Minutes

### Step 1: Get Your API Key

1. Go to https://platform.openai.com/api-keys
2. Sign in or create an account
3. Click "Create new secret key"
4. Copy the key (starts with `sk-`)

### Step 2: Configure the Editor

1. Open the Zeus Editor (`/editor`)
2. Click **"API Key"** button in the toolbar
3. Paste your API key
4. Click **"Save & Test"**
5. Wait for "✓ API key validated" message

### Step 3: Start Using AI Features

- Write some text in the editor
- Click **"Analyze"** to get suggestions
- Click **"Enhance Text"** → Choose enhancement type
- Apply suggestions from the right panel

---

## 💡 Quick Tips

### Enhancement Types

- **Fix Grammar**: Correct spelling, grammar, punctuation
- **Improve Clarity**: Make text easier to understand
- **Make Professional**: Business-appropriate tone
- **Make Casual**: Friendly, conversational tone
- **Make Concise**: Remove unnecessary words
- **Expand Text**: Add detail and examples
- **Make Creative**: More engaging and creative

### Using Suggestions

- Click **"Apply"** to use a suggestion
- Click **"Dismiss"** to ignore it
- Hover over suggestions to see details

### Keyboard Shortcuts

- `Ctrl+S` / `Cmd+S`: Save document
- Regular text editing shortcuts work

---

## 🔧 Troubleshooting

### "Invalid API Key" Error

→ Check your key starts with `sk-`
→ Remove spaces before/after the key
→ Generate a new key if needed

### No Suggestions Appearing

→ Ensure API key is configured
→ Check text is not empty
→ Try clicking "Analyze" again

### Features Disabled

→ Click "API Key" button
→ Enter your OpenAI API key
→ Complete validation
